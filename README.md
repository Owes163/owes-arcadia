## 🌌 About the Project

**Owes Arcadia** is a futuristic, interactive animated website built with **React + GSAP**.
It delivers a **cinematic scrolling experience** combined with **audio-visual effects** and **modern UI design**.

> 💡 Designed to showcase advanced frontend animation skills and immersive UX.

## ⚡ Tech Stack

* ⚛️ React
* 🎞️ GSAP (GreenSock Animation Platform)
* 🎯 ScrollTrigger
* 🎨 Tailwind CSS
* 🔊 Audio Integration
* 🧩 Component-Based Architecture

---

## ✨ Features

✔ Smooth scroll-based animations
✔ Advanced GSAP timeline control
✔ Interactive UI with sound effects
✔ Responsive across all devices
✔ Clean and modular component structure
✔ Modern grid-based layout (Bentley Inspired)

---

## 🛠️ Installation

```bash
git clone https://github.com/Owes163/owes-arcadia.git
cd owes-arcadia
npm install
npm run dev
```
